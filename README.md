# GameCafe-Slot-Management-System-in-C
A console-based C project for gaming lounges featuring admin authentication, slot conflict detection, and automated rent calculation. Supports CRUD operations for slot bookings and persistent file-based storage. Developed as a CSE115L course project at North South University.

# Features
* Admin login & management (change password, add new admin)
* Slot booking management
* Booking conflict detection
* Search, edit, and delete bookings
* Show all bookings with total rent
* Display available game list
* Persistent storage using text files

# Project Structure
* main.c - Entry point and menu handling
* gameCafe.h - Function declarations and structures
* gameCafe.c - Core logic and functionality

# How to Run
On Windows:
1. Compile
Write in the terminal without "": "gcc main.c gameCafe.c -o AnyNameYouWant"
3. Run
Write in the terminal without "": "./TheNameYouWroteInThePreviousCommand"

or directly run the GameCafeProjectExecutableFile.exe.
Note: Keep all the files in the same folder including the .txt files.

# Notes
* Data is stored in .txt files (not encrypted).
* Fully offline and console-based.
* It is a very basic game cafe slot management system
