# GameCafe-Slot-Management-System-in-C
A console-based C project for gaming lounges featuring admin authentication, slot conflict detection, and automated rent calculation. Supports CRUD operations for slot bookings, persistent file-based storage. Developed as a CSE115L course project at North South University.

# Features
> Admin login & management (change password, add new admin)
> Book gaming slots with optional VR addon
> Prevent double bookings (time conflict detection)
> Search, edit, and delete bookings
> Show all bookings with total rent
> Display available game list
> Persistent storage using text files
